## SHUAIPUBLOG

> shuaipuBlog是一个多人博客系统，使用开源系统。自己也正在优化和修改完善中; 具有性能优异、简单实用、安全稳定、支持全文检索的特点。

### 技术选型：

* JDK8
* SSH (Spring、SpringMVC、Hibernate）
* 安全权限 Shiro
* 搜索工具 Lucene
* 缓存 Ehcache
* 视图模板 Velocity
* 其它 Jsoup、fastjson、GraphicsMagick
* jQuery、Seajs
* Bootstrap 前端框架
* UEditor/Markdown编辑器
* font-wesome 字体/图标
* 共同进步........
